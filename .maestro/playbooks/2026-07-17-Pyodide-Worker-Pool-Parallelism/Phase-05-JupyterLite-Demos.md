# Phase 05: JupyterLite Demos

This phase packages the driver-side Python API as a wheel, embeds the JS worker-pool glue so it can be loaded from inside JupyterLite's Pyodide kernel (which itself runs in a Web Worker — the pool spawns nested workers), and ships a JupyterLite site with demo notebooks: pool basics, parallel dask computation, and an in-notebook benchmark. The deliverable is a locally served JupyterLite where a user runs `await compute(...)` in a notebook and watches it parallelize.
