# Phase 04: Browser Demo, Playwright Tests, and Browser Benchmarks

This phase brings the pool and the dask scheduler to their real target environment — the browser. It adds a Vite-served demo page (with COOP/COEP headers so SharedArrayBuffer is available), verifies everything works in a real Chromium via Playwright, and produces a browser benchmark report that mirrors the Node one. Consult the playwright-best-practices skill when writing the test suite.
